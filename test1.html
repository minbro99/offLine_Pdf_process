오프라인 신청서 PDF 추출 가이드


  • PDF에 대한 데이터를 OCR화하여 OCR 작업 진행
    1. 목적: PDF에 있는 데이터를 텍스트로 변환합니다.
    2. 방법:
      1. PDF를 OCR(문자 인식) 작업으로 변환합니다.
      2. OCR 소프트웨어나 알PDF의 OCR 기능을 사용하세요.

    OCR 프로세스

  • PDF 페이지에 베이츠 번호 매기기
    1. 목적: PDF 페이지마다 고유 번호를 추가합니다.
    2. 방법:
      1. 알PDF 실행 후 도구 > 일괄편집 > 베이츠 번호를 선택합니다.
      2. 파일 추가를 클릭하여 작업할 PDF 파일을 선택합니다.
      3. 템플릿 추가(+) 버튼을 눌러 템플릿 디자인을 설정합니다.
      4. 설정이 완료되면 베이츠 번호 추가를 확인하고 저장합니다.

    베이츠 번호 매기기 프로세스

  • 데이터 엑셀 정리하기
    1. 목적: 추출된 데이터를 체계적으로 정리합니다.
    2. 방법:
      1. 엑셀 파일을 열고 다음과 같은 열을 만듭니다:
        2. 필수 입력 리스트
        • 이름
        • 동
        • 페이지 번호
        • 자원명
      2. 각 페이지를 필터링하여 자원명을 그룹화합니다.

    엑셀 정리 프로세스스

  • PDF 페이지 추출 및 정리
    1. 목적: 엑셀 데이터를 기반으로 필요한 페이지를 분리합니다.
    2. 방법:
      1. 알PDF에서 구성 > 추출 > 페이지 내보내기(사용자 지정)를 선택합니다.
      2. 엑셀에 입력한 페이지 번호를 참고하여 각 페이지를 추출합니다.

    페이지 추출 프로세스

    ● 해당 파일 정리 후 저장한 페이지와 엑셀과 매칭 진행
